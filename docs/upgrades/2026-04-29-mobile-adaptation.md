# 2026-04-29 Mobile Adaptation

## 背景

SpeechFolio 现有 Web UI 以桌面端高密度工作台为主：固定左侧导航、表格列表、侧边抽屉和宽屏阅读器在手机尺寸下可访问但不够可用。本次升级目标是让产品在手机和窄屏平板上具备完整的浏览、筛选、选择、批量操作、阅读和任务查看能力。

## 范围

- 移动端应用外壳：小屏隐藏桌面 Sidebar，改为底部导航，并为主内容预留 safe-area。
- 列表页：文章列表、任务列表、来源库、来源视频库在移动端切换为卡片式列表。
- 筛选与排序：移动端保留现有筛选、搜索、排序和重置能力，dropdown 菜单改为底部 sheet。
- 批量操作：文章与视频选择、任务批量重跑/取消继续保留，sticky 操作栏避开底部导航。
- 详情和浮层：文章阅读器、任务详情、Claude 分享、确认弹窗在小屏下改为堆叠阅读或 bottom sheet。
- 设置页：通用表单组件在移动端改为单列布局。

## 非目标

- 不重做品牌视觉方向。
- 不改变 API 行为和数据模型。
- 不新增原生 App/PWA 安装能力。

## 验收点

- `390px`、`430px`、`768px`、`1024px` 宽度下主要页面不出现横向页面级滚动。
- 底部导航不遮挡批量操作栏、弹窗按钮和任务抽屉底部内容。
- 文章、任务、来源视频页的筛选、分页、跨页多选、重跑、导出仍可操作。
- 文章详情中上一篇/下一篇、任务面板、Claude 分享、导出功能仍可操作。

## 回滚策略

本次主要是 CSS 响应式规则和文档变更。如移动端出现严重回归，可回滚本升级提交，桌面端主体逻辑不受数据迁移影响。
