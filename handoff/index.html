最短路径

给 Claude Code / Cursor

  1. 把整个 handoff/ 喂给 agent
  2. 让它先读 01-architecture 建骨架
  3. tokens.css + icons.svg 复制进 /public
  4. 按 03-components 顺序落组件
  5. 按 04-pages 拼页面
  6. 用 MSW mock 05-api 跑前端，后端并行

给人类工程师

  1. 先在新窗口开 v3 原型 当对照
  2. 用本页作为索引，按需跳转文档
  3. 有疑问直接回到原型 devtools 看元素
  4. 所有尺寸 / 颜色都用 token，不要写 magic number
  5. Tweaks 面板是内置开发工具，见 07-interactions
