# 原稿：付鹏新书《见证逆潮》：构建基于交易员视角的经济世界观

新的一年里，我毛遂自荐一下自己的《见证逆潮》吧。这本书是由见闻读书和华尔街见闻联合出品，由电子工业出版社出版。这本书是我职业生涯多年形成的经济世界观和思考体系，是我沿着思考路径，比对真实场景，通过对我过去积累的交易员经验进行总结而得到的结论。现在我把自己的经济世界观和思考体系展现给读者，希望能够给读者提供一些切入宏观经济现象，看待资产变化情况的新角度。第一章是导论，我从自己的经济世界观和思考体系出发，总结了我认为最重要的三组关键词：分工和分配、债务和杠杆、收入和贫富。它们分别决定了财富的创造、财富的扩张和财富的分配。这三组关键词是我们理解人、社会、国家行为的重要维度，也是。也是我们后续从中观或微观角度展开论述所有大类资产关系的基础。第二章是对第一组关键词分工和分配的具体阐述。经济学理论的分工和分配的结果有着较为苛刻的前提条件，但现实世界并不存在绝对公平的分工和分配。第三章是对第二组关键词，债务和杠杆的理解。人们在完成财富的原始积累后，让其滚雪球般增长需要依靠债务和杠杆，但折现未来的特性决定了债务和杠杆的两面性。第四章是对二战后充满竞争的世界的总结。二战后，世界形成了新的分工和分配秩序。第五章是对全球货币体系变化和里根大循环的介绍。在新的分工和分配秩序下，全球货币体系不断变革，并形成了美元霸权。而著名的里根大循环，则建立在美元作为全球货币的基础之上。第六章是对全球化程度加深过程中，不同经济体在全球分工中所处的地位的理解。这是我评价一个经济体、了解其经济结构、决定其分析方式的基础。通过拆分全球经济一体化下的三级模型，读者可以对全球各个经济体的角色有一个初步的了解。第七章探讨了在全球化经济一体化下，各国金融货币政策与利率病的状况，这是对债务杠杆的载体金融的扩展和反馈。第八章是本书的核心，也是大多数读者较为关心的一章。F I C C 大类资产的概念进入我国仅十多年时间，但对其研究的程度逐渐加深。在经过海外对冲基金交易的训练后，我认为。我认为 F I C C 的核心宏观是是全球分工分配的理解，而微观源自于利率曲线，利率曲线可以包含的信息最多，想要透彻掌握也最难，而我们对跨资产的理解和思考，离不开对利率曲线的解读。因此，本章在利率、信用、债务、财政、汇率、商品、权益的理解和它们相互之间的联系上着墨尤多。第九章浅谈了一下我对中国债务的看法，第十章是对我的经济世界观中的最后一组关键词，收入与贫富的展开。由于经济全球化和金融杠杆，贫富分化有所加重，本章详细分析了日本、中国、香港和美国社会的推演演化史，并对房地产消费和资本垄断等失衡问题进行了分析。第十一章和第十二章是对当前经济环境变化的一些思考，也是本书名为见证逆潮的来源。贫富分化加剧，逆全球化进程加速，其所产生的宏观意向超出了过去四十多年市场的一般认知，而我们有幸成为其见证者。第十二章着重分析了日本长短利率倒挂的曲线及其背后的深刻原因，并延伸了一些相关的思考。第十三章是对未来可能性的探究。这本书是我对自己的经济世界观和思考体系的总结，也是我对自己过去经验的总结。我在保证数据客观、真实、准确的情况下，向读者展示了我对经济的思考。看待问题的方式有千千万万种，希望本书的内容对各位读者有所启发。现在图书的购买链接已经放在了左下角的橱窗中，由出版社直接邮寄，有感兴趣的朋友可以直接点击购买。
