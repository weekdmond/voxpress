# 重写版：通胀挣钱，通缩花钱：一个基准财务原则

## 通胀挣钱，通缩花钱：不是口号，是生存基准

这不是选择题，而是全球资本运行与个体生命周期的真实逻辑：**在通胀的地方挣现金流，在通缩的地方花购买力；违背它，就必然陷入资产负债表的恶性三角**。

有人传付鹏可能去海外退休养老，必须澄清：作为一个在全球投资和做交易的人，怎么可能违背这个最基本的原则？从2018、2019年起，国内与海外宏观环境已清晰分化：
- 中国进入价格战主导的广义通缩
- 欧美（尤其美国）在疫情后经历显著通胀，服务业、物价均飞涨

**所以我的判断从未动摇：老钱在通胀中完成积累，在通缩中兑现消费。我怎么可能去通胀高企的海外养老？**

## 风险偏好随年龄下降，是生理现实，更是财务理性

手术之后，风险偏好进一步降低。从去年起，老花眼、肩周炎陆续出现；今年手术后，连刷手机都费劲，更别说追小姐姐了。现在迷恋的是上海的足疗：

> 存一万送三千，白天场不到两百块，比几年前北京"拇指生"打骨折还便宜

这背后是宏观现实：**国内服务业全面卷入价格战**。但你要问——享受服务的人，赚钱到底为了什么？当然是购买商品和服务。那些"挣钱只为挣钱"的人，在付鹏看来心理上有点疾病。

人生难得来一次，核心目的就是消费。所以当通缩来临、购买力增强时，对已完成资本积累的老钱而言，正是花钱养老的最佳窗口——回趟成都，早餐两个包子加豆浆，下午公园溜达，晚上捏个脚，整体成本极低。

## T字表格：每个人的财务思维底层框架

必须永远保持一个**T字表格思维**——即资产负债表意识。这是教科书最基础、却最被忽视的金融思维：

- **资产端**：你掌握的生产资料及其回报——劳动力（体力/脑力）、资本、土地、权力
- **负债端**：你的支出、借贷、消费、义务

> 万事万物皆具两面性：通胀时资产回报率高，利率自然高；通缩时资产回报率差，利率才低。利率从来不是"因"，而是"果"——它是经济冷暖的体温计，而非遥控器。

很多人误以为"利率低=该借钱投资"，一听就是没进过社会、没跟银行打过交道。银行放贷看抵押、看流水、看稳定性——利率低，对无产者毫无意义；它只对已有信用、有抵押、有稳定收入的人构成真实机会。

**本质是：经济好→回报高→利率高；经济差→回报差→利率低。**

日本的例子印证了这一点：利率足够低了，但它必须先消化债务、重建信心、找到新增长点，低利率才能真正传导效果。

## 好通胀与坏通胀：收入增速是分水岭

分析通胀通缩，必须锚定一个基准：**全世界绝大多数人的核心生产资料，就是你自己——你的劳动力。**

改革开放后那一代人为什么享受"好通胀"？
- 收入在涨
- 早买房
- 房价在涨
- 几乎无负债
- **结果：资产与收入双增值**

而2012年后很多人觉得"坏通胀"，因为房价涨幅长期大幅跑赢工资——不是收入不涨，而是资产门槛太高。

**关键认知：通胀下你只是有压力；通缩下你才真危险。** 尤其当你同时面临：
- 收入增速放缓甚至失业
- 手持高价房产（资产端通缩）
- 背着房贷利息（负债端刚性通胀）

> 这就是最差组合：现金流恶化 + 资产贬值 + 负债刚性。利息本质是通胀，而房价下跌是通缩——三者叠加，资产负债表瞬间坍塌。

## 最优财务组合：野比大雄的套息交易案例

理想状态是什么？倒过来：
- 现金流好（收入高速增长）
- 资产通胀（如人民币房产、股权）
- 负债通缩（如日元低息贷款）

这就是金融里的**carry trade（套息交易）**的生活化表达。

**范本故事：野比大雄去中国**

1993年，假设野比大雄放弃日本20万日元月薪，赴上海外企工作：

| 指标 | 数字 |
|------|------|
| 上海月薪 | 3000美元 |
| 汇率 | 1美元 ≈ 8.3-8.4人民币 |
| 折合月薪 | ≈2.5万人民币 |
| 当时静安区房价 | 1500美元/㎡ |
| 100㎡房产成本 | 15万美元 ≈ 150万人民币 |
| **工作月数即可购房** | **约6个月** |
| 至今房产价值 | **>1500万人民币** |

持有这套房产至今，升值超10倍。**若再借1%日元贷款投入中国资产——就是典型的"负债通缩 + 资产通胀 + 现金流增长"黄金三角。**

相比之下，卖掉北京四合院赴美打工的老一代华人，踩的是反向陷阱：
- 卖出通胀资产（北京房）
- 换回缓慢增长的劳务收入
- 全球化曾用低价商品补偿其购买力
- 但当美国通胀重启、非法移民收紧、服务业人工涨价，补偿消失
- **2008年次贷危机就是遮羞布被撕开的时刻**

## 跨国配置的本质：把钱放在通胀处，把人放在通缩处

过去四十年最优解是什么？

- **老公**在国内企业或资本市场挣钱（通胀处现金流）
- **妻子孩子**在海外享受福利与低价服务（通缩处消费）
- 同时持有**中国房产+美股股权**（双资产通胀）
- 若在跨国科技公司任职，还叠加**股票期权**（收入增长+资产通胀）

**今天呢？** 香港老人卖楼去珠海养老的现象就是活生生的"通胀挣钱，通缩花钱"：
- 深圳湾一小时船程
- 养老金照领（通胀处的现金流保障）
- 足疗按摩打折，菜市场新鲜便宜（通缩处的购买力增强）
- 这不就是完美的国际化配置？

## 财务真相的三个维度

每个人必须评估自己的情况：

1. **现金流**：你的收入是否增长？有无失业风险？
2. **资产**：你持有的资产是通胀还是通缩？
3. **负债**：你的债务是通胀还是通缩？

能否做跨国carry trading组合？把这些放在一起，才是属于你自己的财务真相。

## 结语：这不是理论，是已经发生的历史

今天唠的这些，不是假设，而是：
- 野比大雄们真实走过的路
- 四合院主人错失的窗口
- 香港老人正在实践的生活

**理解它，你才能在通胀中安心挣钱，在通缩中踏实花钱——不焦虑，不盲从，不陷坑。**
