# 原稿：垃圾资讯泛滥比市场波动更致命

做投资其实最忌讳的就是看，哎呦，谁看对了，谁看错了。你知道在赌场的时候就有那种人呢，这个人三连庄了，四连庄了，我在背后跟着他，最忌讳的就是这个。现在的工具其实比我们当年多得多了，无论是你可以收集到的信息数据库，包括你可以看到的这些报告。你想二十多年前的时候，我们获得这些信息和资讯难度是非常的大的。同样的这个道理，其实大部分人在网上收集这类信息的时候，我一直强调的是你要有一个学习的方法，你要利用这些工具去进行信息的处理，同时呢去筛选出这种逻辑线。你举个例子，研究报告不要看结论，哎呀，我碰到过那种研究报告就看结论，然后呢去市场上看涨了还是跌了，我来印证这个报告对和错，这是大错特错了。你看好多的媒体喜欢干嘛呢？什么大摩明年上涨看到多少点，高盛什么东西看空到多少点，它是媒体你们懂吧？媒体老想去抓眼球，其实我说年轻人不要看这个东西，我说你们要看完整新闻，不要老看那个标题弹出，你拿标题弹出去对。你的资金负责，那不是扯淡嘛？所以说你要看原文，而且要看逻辑，比如说一二三四，a b c d，它的原因是什么？那我们逐条去分析这些原因，我如果认可了，那我应该策略怎么去做？我不认为现在的资讯短缺，恰恰我觉得现在的资讯是过剩的，就是垃圾资讯太多了。举个例子，比如说就有人刻意的会潜伏到各种各样的投资群里，然后完了以后再去在 Twitter 上大量的散布，据传怎么怎么样，据说怎么怎么样，然后完了以后我是不是就可以截图扔回到各种群里？你会发现大部分人有辨别能力吗？没有。甚至我当时真的是一个朋友转给我 X 的一个截图，然后问我说付总你觉得这个东西怎么看？我说这是谁发的？他说我也不认识。我说他一不是新闻媒体，第二他也不是加 V 的官号，你连他是谁你都不知道，然后你来问我他是真的假的？甚至有一个研究员他发我了一个路透新闻的一个页面，说哎付总，路透发了这么个新闻，你怎么去看？我首先第一点我肯定不会怀疑说这个新闻是假的，因为你。截的是那个窗口页面，这个应该是真的。哎，但那天呢就比较巧，我正好就在终端前坐着，然后我说那你等会儿，我看一下。我就啪啪啪，我一搜，我竟然没找到。然后呢我就说，我说这个什么情况，难道是我这个搜索检索的有问题？我搜了很多，没找到。我说有点不对，我没法确认这个新闻。我说那你从哪儿来的呢？他说，哎，我在外网X上截的，有个什么博主发的。我说那博主是谁呢？他给我发过来什么什么七七八怪的这种名字，我靠，我说这玩意儿有没有可能是假的呢？P的呢？我说我没法分辨。所以说那就你看类似于这种新闻的传播影响很大。去年前年监管部门当时严打小作文，很多人光以为小作文就是那个所谓的什么八卦消息啊，上市公司的什么传闻啊，不，现在有很多的信息都是从外网倒灌到你的微信群里的，尤其投资圈更是这样。然后大量的记者、媒体编辑也都混迹于各种群，然后会把各种据传八卦当成新闻。新闻是什么？你问问那些老。记者，记者的底线是什么？我写一篇报告的时候，我要有来源，要有引用，要有自己的调研，对吧？要有自己的采访，才能够写一篇报告。现在呢，根本不是，所以这就导致看似信息快了，信息多了，但是也坏了啊。所以说呢，我们千万别干一件事情，就是去那个各种投资群里看八卦小作文，那就嗝屁了。我之前跟很多朋友说，我说你要想知道我的观点，简单呀，我写日记啊，那么长的文章你看呀，对吧？你不要说把付总的这个呃音频或者视频就中间剪一小块，然后扔出来，他就说哎付总你是不是这个观点？我说哎呀，你能不能把全文看完？那每一句话前因后果，那个答案是不一样的。然后呢，我们很多人就愿意看那种弹出来就一句一句一句一句，上面写个什么暴跌了，你应该怎么干？暴涨了，你应该怎么干？我靠，实际上一看百分之二叫暴涨暴跌，这个就就很要了命了，就是大家为了流量拼命的在抓热点啊，所以这点还是奉劝大家千万别用。这种信息去做你的投资决策。
