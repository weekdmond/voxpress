# 付鹏视频稿生成文章忠实度评估报告

- 评估对象：`exports/fupeng_claude_review_pack/articles` 下 523 篇文章
- 评估方式：逐篇定量比对"原稿 vs 原始生成文章"，产出 0-100 的忠实度分数 + 诊断
- 交付：
  - 《评估总表.csv》——全量 523 篇分数+诊断（按序号排序，同 README 索引）
  - 本报告——方法、分布、重点篇目清单

## 一、方法论

### 1.1 为什么不能只看长度
付鹏的内容是直播/访谈逐字稿。原稿里常有两类"可以删"的内容：
- **口头禅与填充**：啊/嗯/对吧/这个/那个/我说呢……
- **直播现场与广告**：抽奖、签名杯、读书会报名、喊小助理、念粉丝评论、感谢赞助等

因此同一个经济学论点，原稿 2 万字被压成 1500 字的"生成文章"也可能是合理的；
但一个 37 字的新春祝福被扩写成 1000 字的长文则几乎一定是虚构。

### 1.2 分层评估
按原稿字数分三档，采用不同权重与容忍度：

| 层级 | 原稿字数 | 数量 | 评估重点 |
| --- | --- | --- | --- |
| 短稿 short | <300 | 11 | 生成文章不应"无中生有"——扩写超过 4 倍即有虚构风险 |
| 中稿 medium | 300–3000 | 433 | 核心论点、数字、人名、案例应基本保留；压缩到 1/3 以下需要核查 |
| 长稿 long | >3000 | 79 | 允许大幅压缩，但"高频关键词""硬数字"应覆盖 ≥45% |

### 1.3 指标
对每篇文章计算：
- **关键词保留（key_keep）**：原稿中词频 ≥3（或 ≥2，视关键词数量）的内容词，在生成文章中出现的比例
- **内容词保留（content_keep）**：原稿所有去停用词后的名/动/形/专有名词集合，在生成文章的覆盖率
- **数字/年份保留（num_keep）**：原稿中所有可识别的年份/金额/百分比/倍数/岁数，在生成文章中的命中率（已对中文数字"零八年→2008年"做归一化）
- **ASCII 术语保留（ascii_keep）**：英文品牌/缩写/技术词的保留率（A股、KPI、LV、CAPE 等）
- **压缩比（ratio）= 生成字数/原稿字数**
- **虚构风险（fabricate_risk）**：原稿 <200 字且生成 >400 字时按差值打分，≥0.5 视为高危

综合为 **忠实度分（faith_score）**，满分 100：
- 短稿：关键词 25% + 内容词 35% + 数字 10% + ASCII 5% + 长度 25%
- 中稿：关键词 35% + 内容词 30% + 数字 15% + ASCII 10% + 长度 10%
- 长稿：关键词 45% + 内容词 20% + 数字 20% + ASCII 10% + 长度 5%
- 若 `fabricate_risk > 0.3`，总分再乘以 `(1 − 0.6·风险)`

### 1.4 分数的解释

| 分段 | 解读 |
| --- | --- |
| ≥90 | 良好：核心观点、数字、案例基本完整 |
| 80–89 | 基本合格：有轻量漏点但不改主旨 |
| 70–79 | 中等：建议人工复核 |
| 55–69 | 偏弱：存在关键漏点或压缩过度，建议复核或重写 |
| <55 | 差：核心观点丢失或大面积虚构，建议重写 |

### 1.5 局限（基于抽样人工复核的修正）
在 5 个分段各抽 2 篇 + 所有高危篇目做了人工复核，证实评分系统存在以下偏差：

- **虚构检测可靠**：162 / 226 / 228 等 short→long 扩写篇的虚构判断全部命中
- **高分段（≥85）基本可信**：抽查的 522/164 等高分篇确实完整保留了原稿观点、案例、数字
- **中稿偏高风险**：当生成文章压掉了具体案例（人名/地名/实物如"华润物业""上海四川北路"）
  但保留了框架术语时，关键词保留率仍然较高，分数会偏高 5–15 分。例如 #95《逆周期现金流才是财富真正的护城河》
  实际丢失了 3 个具体案例，但自动分 77.1 看起来还可以；真实质量应当是"偏弱 60–65"
- **长稿偏低风险**：当原稿含大量闲聊/广告（送行寒暄、员工对话等）时，生成文章把它们删干净是对的，
  但这些闲聊用词也会被算到"关键词"里造成低分。例如 #125《付鹏重返越南》自动分 50.1，
  但人工复核发现**删除的主要是闲聊，论证、案例、数字基本保留，真实质量接近 75 分**
- 定量指标不能直接判断"观点是否被歪曲"，只能识别"组成元素是否缺失"
- 中文分词难免漏切词，会造成个别误报

**结论：本报告分数作为"分诊"信号使用，最终复核时须注意上面两类系统性偏差。**
