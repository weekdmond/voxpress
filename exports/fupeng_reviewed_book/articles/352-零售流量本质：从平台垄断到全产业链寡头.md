# 零售流量本质：从平台垄断到全产业链寡头

## 流量的底线与分配权

为了获取流量，很多底线是会被触碰的。我们开玩笑说，获得流量最简单的方法是什么？裙子短一点，腿拉长一点，你的流量自然就大了，这就是“三俗”。但很简单，这种东西对社会的影响危害极大，因为所有人都是逐私、逐利的，都奔着这个去了，这能不监管吗？

第二，流量的分配一定是均衡的吗？恐怕不一定。我估计很多人已经发现了这个问题。流量既然是钱，那没有人会随便给你的。

包括以前互联网平台的网页时代，也有这个问题。当年不是抓过很多“店小二”吗？为什么？因为店小二掌握的是流量密码。我给你家和不给他家，你俩的销售业绩差距很大。很简单，我也可以拿这东西入股来分钱。所以在杭州，很多懂的人都知道，买豪宅的有不少是店小二，原因就在这儿。

## 平台并非真正的共享经济

到现在为止，情况其实是一样的。平台本身的目的是通过一个工具让大家在上面发布内容，然后它用这些内容实现了所有的流量聚集。

理论上来讲，你用这个内容实现的所有流量，是不是应该分给所有的人？这才叫共享。然而并不是。你们所有贡献出来的流量，最终的决定权可能在平台，也可能在某些人手里。

> 这里边真的是通过大数据去算吗？怕也不是吧。

所以说，当有几个大的 IP 出来掌握着大量流量的时候，零售就会主动地切过去。至于用户真的从中间获得了很大的收益吗？我可以跟大家讲，并不是。

真正做到无私的，才会有真正的厂家到用户、到 To C 端的便宜。否则的话，它只是商业竞争的一种手段。当这种手段完成以后，要么就是从 C 端吃，要么就是从生产端吃。

## 线上线下核心逻辑毫无区别

大概这几年，当这种情况一旦形成，商家也是苦不堪言的，就跟当年线下一样。

我这个品牌家居，如果不进到你这个卖场里，我就没法混了。因为全国各地都是你的卖场，全国各地老百姓买这个东西都奔你那儿去，我不进去就根本没法做生意。那问题也就来了：我的成本代价会很高，我赚的钱会非常辛苦和艰难。

而 C 端一定会获得利益吗？不可能的，除非中间一毛钱不挣。但事实上来讲可能吗？不可能。刚才我已经讲过了，流量就是财富。

按照以前线下的逻辑，从广告费到上架费用，再到资金沉淀，到后端的营销分成，你这所有的钱其实中间环节都吃掉了。整个模型没有区别，线上线下核心逻辑都没有区别。

无非就是从线下小型化到平台化，从电视机到手机，从线下平台寄托于房子、区域位置的物理界限，到网上非物理性界限，仅此而已。所以说，本质问题其实都一样。

## 终极形态：全产业链寡头与 IPO

终极形态是什么？刚才我们讲的还比较偏实体。你可以看到从生产端到中间的销售、广告、批发、零售、获客，甚至包括仓储物流供应链发货，这些都还比较偏实体。

你赚的钱也是这个道理：你要么赚广告费用，要么赚渠道费用，要么赚销售提成，要么赚资金沉淀。还有什么？

干到最后，理论上是入股上面的生产厂家。换句话说，这个游戏到最后一定是：我跟合作好的这个厂家，我要把它吃下来。

倒过来我就会全产业链吃。我最后吃到生产厂家，变成我自己的生产厂、我的品牌，然后以我极其强大的影响力、足够强大的销售渠道，去实现这一环。形成闭环以后干嘛呢？IPO。

> 但是这种情况下就会形成非常典型的寡头垄断。

甚至到最后我们会发现，某一类产品只有他们家能做，别人家根本做不了。为什么？因为你们做都卖不出去。

一旦形成这种东西，请问消费者你还能获得便宜的吗？当然不会了。所以说，有些事情看明白就好了。当然了，这还是生产，如果再搞一个 IPO，你们可以想想会是个什么概念。
